# Agents for Urban Planning

Course tutorials and reference materials for building agentic workflows in urban-planning tasks.

## Structure
- `tutorials/`: hands-on tutorials (each tutorial has its own detailed README)
- `docs/`: plans, notes, and supporting documentation
- `agents/`: shared agent-related assets at repo level

## Recommended Start
1. Open `tutorials/01_city_gdp_collection/`.
2. Follow the setup/run instructions in that folder's `README.md`.
3. Use `tutorial-1.ipynb` for the guided workflow walkthrough.

## Notes
- API keys are expected via local `.env` files where required.
- Keep tutorial-level execution from each tutorial folder to avoid import/path issues.
