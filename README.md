# Agents for Urban Planning

Course tutorials and reference materials for building agentic workflows in urban-planning tasks.
DATE: 2026-03-05
Author: Yuan

## Structure
- `tutorials/`: hands-on tutorials (each tutorial has its own detailed README)
- `docs/`: plans, notes, and supporting documentation

## Prepare for the class
- Bring a laptop with access to Google Colab (as long as you have a gmail account)
- Register an account for any coding agents (Claude Code, Codex or Gemini)
- Email `yuan@ready.net` with subject `Urban AI Seminar` to get an OpenAI API
- Go to `https://www.tavily.com/` to register a free account and get an API (up to 1000 search free)
- [Optional] get a Github account
- Link to Tutorials: `https://github.com/brookefzy/agents-for-urban-planning/tree/main`

## Notes
- API keys are expected via local `.env` files where required.
- Keep tutorial-level execution from each tutorial folder to avoid import/path issues.
