# Output Comparison Rubric (Tutorial 2)

Use this rubric to compare outputs from:
- Baseline prompt
- Research-grade prompt
- Skill-guided prompt

Score each category from `1` (poor) to `5` (strong).

## Rubric
- **Schema completeness**: Are required fields present and consistent?
- **Geography correctness**: Is city-level GDP distinguished from country GDP?
- **Source quality**: Are authoritative sources prioritized?
- **Evidence traceability**: Does each row include a usable evidence snippet + URL?
- **Uncertainty labeling**: Are ambiguous rows clearly marked `inReview` or `missing`?
- **Method transparency**: Are assumptions and failure modes documented?

## Discussion Prompts
- Which version is fastest to produce?
- Which version is easiest to audit?
- Which version would you trust for a planning memo draft?
- Which version is closest to a research-grade workflow?
